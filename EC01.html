Dénombrements

Compter les éléments d'un ensemble vérifiant une propriété

Principe fondamental DEF

Avant de compter, toujours se poser la question : l'ordre intervient-il ?

OUI, l'ordre compte
Arrangement
Désignation de postes distincts, tirages successifs sans remise
NON, l'ordre ne compte pas
Combinaison
Choix d'un sous-ensemble (personnes, cartes, boules...)

Arrangements DEF

Un arrangement d'ordre p parmi n éléments est un sous-ensemble ordonné de p éléments pris parmi n.

Apn = n × (n−1) × (n−2) × … × (n−p+1) = n! / (n−p)!
Cas particulier — Permutations : Quand p = n (on ordonne tous les éléments) :
Pn = Ann = n!

Combinaisons DEF

Une combinaison d'ordre p parmi n éléments est un sous-ensemble non ordonné de p éléments.

Cpn = Apn / p! = n! / [p! × (n−p)!]

On dit aussi : « p parmi n » ou coefficient binomial. Propriétés utiles :

  • C0n = Cnn = 1
  • Cpn = Cn−pn (symétrie)
  • Cpn + Cp+1n = Cp+1n+1 (triangle de Pascal)

Formule du binôme de Newton THM

(a+b)n = Σp=0→n Cpn · an−p · bp

Cas particulier : en posant a = b = 1 → 2n = Σ Cpn (somme de tous les coefficients binomiaux de la ligne n).

⚡ Règle de décision rapide

→ Rangement, désignation, tirage successif sans remise : Arrangement
→ Ordonner tous les éléments : Permutation

Exemple 1 — Bureau de 27 personnes EX

Combien de façons de choisir un président, vice-président, secrétaire et trésorier parmi 27 personnes ?

L'ordre intervient (4 postes distincts) → Arrangement A427
N = A427 = 27 × 26 × 25 × 24 = 421 000

Exemple 2 — Jeu de 32 cartes EX

Combien de jeux de 5 cartes peut-on former dans un jeu de 32 cartes ?

L'ordre n'intervient pas → Combinaison C532
C532 = (32×31×30×29×28)/(5×4×3×2×1) = 201 376

Exemple 3 — 5 chiffres avec {1,2,3,4,5} EX

Combien de nombres à 5 chiffres distincts peut-on former avec {1,2,3,4,5} ?

On ordonne tous les 5 éléments → Permutation P5
N = P5 = 5! = 5×4×3×2×1 = 120

Espaces Probabilisés Finis

Univers, événements, probabilités et leurs propriétés

Vocabulaire fondamental DEF

  • Univers Ω : ensemble de tous les résultats possibles d'une expérience aléatoire.
  • Événement : toute partie (sous-ensemble) de Ω. Noté P(Ω) l'ensemble de tous les événements.
  • Événement élémentaire : événement à un seul élément {ω}.
  • Événement impossible : ∅ (ensemble vide).
  • Événement certain : Ω lui-même.
  • Événement contraire de A : Ā = Ω \ A (complémentaire de A).

Opérations sur les événements PROP

A ou B
A ∪ B
Au moins l'un des deux est réalisé
A et B
A ∩ B
Les deux sont réalisés simultanément
Événements incompatibles : A et B sont incompatibles si et seulement si A ∩ B = ∅
Lois de De Morgan : Ā∪B̄ = Ā∩B̄ et Ā∩B̄ = Ā∪B̄

Définition d'une probabilité DEF

Une probabilité sur Ω est une application P : P(Ω) → [0,1] vérifiant :

  • P(Ω) = 1
  • Si A ∩ B = ∅ alors P(A ∪ B) = P(A) + P(B)
P(Ā) = 1 − P(A)  |  P(∅) = 0

Équiprobabilité THM

Si tous les événements élémentaires ont la même probabilité (univers fini de cardinal n) :

P({ωᵢ}) = 1/n   et   P(A) = card(A) / card(Ω) = cas favorables / cas possibles

Théorème général — P(A∪B) THM

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Si A et B sont incompatibles (A ∩ B = ∅), on retrouve P(A ∪ B) = P(A) + P(B).

Probabilité conditionnelle DEF

La probabilité de B sachant que A est réalisé (P(A) ≠ 0) :

PA(B) = P(A ∩ B) / P(A)
Formule des probabilités composées : P(A ∩ B) = P(A) × PA(B)

Événements indépendants DEF

A et B sont indépendants si et seulement si :

P(A ∩ B) = P(A) × P(B)

Conséquence : si A et B sont indépendants, alors PA(B) = P(B). La réalisation de A n'influe pas sur B.

Exemple — Jeu de cartes (tirage simultan) EX

Jeu de 32 cartes, on tire 4 cartes. Calculer P("2 rois et 2 valets tirés").

card(Ω) = C432 = 35 960

P(A) = 36/35960 = 9/8990 ≈ 0,001

Exemple — Urne, tirages successifs sans remise EX

P(A₁∩A₂) = P(A₁)×PA₁(A₂) = (5/8)×(4/7) = 20/56 = 5/14 ≈ 0,357

Variables Aléatoires Discrètes

Loi de probabilité, espérance, variance, fonction de répartition

Variable aléatoire DEF

Une variable aléatoire X est une application de Ω dans ℝ : à chaque résultat ω on associe un nombre réel X(ω).

Discrète d'ordre fini : X prend un nombre fini de valeurs x₁, x₂, …, xₙ, avec pᵢ = P(X = xᵢ).

Espérance mathématique DEF

Analogue de la moyenne statistique pondérée par les probabilités :

E(X) = Σ pᵢ · xᵢ = Σ xᵢ · P(X = xᵢ)

Propriétés :

  • E(aX + b) = a·E(X) + b
  • E(X + Y) = E(X) + E(Y)
  • Si X et Y indépendantes : E(X·Y) = E(X) × E(Y)

Variance et écart-type DEF

V(X) = E[(X−E(X))²] = E(X²) − [E(X)]²
σ(X) = √V(X)

Propriétés :

  • V(aX + b) = a²·V(X)
  • Si X, Y indépendantes : V(X±Y) = V(X) + V(Y)

Fonction de répartition DEF

F(x) = P(X ≤ x). C'est une fonction en escalier croissante pour les variables discrètes.

  • F(x) = 0 pour x < min des valeurs de X
  • F(x) = 1 pour x ≥ max des valeurs de X
  • P(a < X ≤ b) = F(b) − F(a)
  • P(X ≥ a) = 1 − F(a⁻) = 1 − P(X < a)

Exemple — Deux pièces de monnaie EX

On lance 2 pièces non truquées. X = gain du joueur (10F par pile). Distribution :

X01020
P(X=x)1/41/21/4

V(X) = E(X²) − [E(X)]² = (0+50+100) − 100 = 50
σ(X) = √50 ≈ 7,07

Loi Binomiale & Loi de Poisson

Variables aléatoires discrètes particulières

Épreuve de Bernoulli DEF

Une épreuve de Bernoulli est une épreuve à deux issues seulement :

Succès (résultat a)
P(a) = p
Échec (résultat b)
P(b) = q = 1−p

Exemples : pile/face, succès/échec examen, machine en panne/en marche, etc.

Loi binomiale B(n,p) DEF

On répète n fois la même épreuve de Bernoulli de manière indépendante. X = nombre de succès.

P(X = k) = Ckn · pk · (1−p)n−k    k ∈ {0, 1, …, n}
Espérance
E(X) = np
Variance
V(X) = npq
Écart-type
σ = √(npq)

Loi de Poisson P(λ) DEF

X suit la loi de Poisson de paramètre λ > 0 si :

P(X = k) = e−λ · λk / k!    k ∈ ℕ
Espérance
E(X) = λ
Variance
V(X) = λ
Écart-type
σ = √λ
Champ d'application : Phénomènes où le futur est indépendant du passé : pannes de machines, appels téléphoniques, files d'attente, mortalité, sinistres...

Approximation B(n,p) ≈ P(λ) THM

Quand n est grand, p proche de 0 et np pas trop grand, on peut approcher la loi binomiale par la loi de Poisson :

B(n, p) ≈ P(λ) avec λ = n·p
Conditions pratiques : n ≥ 30 et p ≤ 0,1 et np < 15
ou n ≥ 50 et p ≤ 0,1 et np ≤ 10

Exemple — Loi binomiale (examen) EX

Prob. d'admission = 0,3. 5 candidats. P(aucun admis) = ?

X ~ B(5 ; 0,3)
P(X=0) = C⁰₅ · (0,3)⁰ · (0,7)⁵ = (0,7)⁵ ≈ 0,168

P(au moins 2 admis) = 1 − P(X=0) − P(X=1) = 1 − 0,168 − 0,360 ≈ 0,472

Exemple — Loi de Poisson (guichet) EX

En 10 minutes, 5 clients en moyenne arrivent à un guichet. Prob. d'avoir au moins 8 clients ?

X ~ P(λ=5)
P(X≥8) = 1 − P(X≤7) = 1 − 0,867 = 0,133

Exemple — Approximation de Poisson EX

Sur 10 000 pièces, 50 en moyenne sont défectueuses. Parmi 400 pièces, probabilité qu'une seule soit défectueuse ?

p = 50/10000 = 0,005 ; n=400 → λ = 400×0,005 = 2
P(X=1) = e⁻² × 2¹/1! = 2e⁻² ≈ 0,271

Variables Continues & Loi Normale

Densité, loi de Gauss, ajustement, échantillonnages

Variable aléatoire continue DEF

X est continue si X<Ω> est un intervalle de ℝ. La probabilité d'un singleton est nulle : P(X=x) = 0 pour tout x.

Conséquence : On ne caractérise plus X par des couples (xᵢ, pᵢ) mais par sa fonction de répartition F ou sa densité de probabilité f.

Densité de probabilité DEF

f est la densité de X si f est la dérivée de F sur ℝ. Propriétés :

  • f(x) ≥ 0 pour tout x réel
  • ∫−∞+∞ f(t)dt = 1 (aire totale = 1)
  • P(α ≤ X ≤ β) = F(β) − F(α) = ∫αβ f(t)dt
E(X) = ∫−∞+∞ t·f(t)dt    V(X) = ∫−∞+∞ [t−E(X)]² f(t)dt

Loi normale N(m, σ) DEF

X suit la loi normale (ou de Gauss-Laplace) de paramètres m et σ si sa densité est :

f(t) = 1/(σ√2π) · exp[−½·((t−m)/σ)²]
Espérance
E(X) = m
Variance
V(X) = σ²
La courbe est en « cloche », symétrique par rapport à x = m. Points d'inflexion en x = m ± σ.

Loi normale centrée réduite N(0,1) THM

Si X ~ N(m, σ), alors T = (X−m)/σ suit la loi N(0,1).

On calcule les probabilités grâce à la table de la fonction Π (ou Φ) :

  • P(T ≤ t) = Π(t)
  • P(T ≥ t) = 1 − Π(t)
  • P(T ≤ −t) = 1 − Π(t) (symétrie)
  • P(t₁ ≤ T ≤ t₂) = Π(t₂) − Π(t₁)

Intervalles remarquables PROP

IntervalleP(m−aσ ≤ X ≤ m+aσ)Signification

Théorème de la limite centrée (TLC) THM

X₁, …, Xₙ variables indépendantes de même loi, d'espérance m et d'écart-type σ. Pour n grand :

S̄ = (X₁+…+Xₙ)/n suit approximativement N(m, σ/√n)
Application : Distribution d'échantillonnage des moyennes.
σ(X̄) = σ/√n (tirage avec remise)  ou  σ/√n · √((N−n)/(N−1)) (sans remise).

Approximation B(n,p) ≈ N(m,σ) THM

Si n est grand et p ni trop proche de 0 ni de 1 :

B(n,p) ≈ N(m=np, σ=√(npq))   si n≥30 et 0,2≤p≤0,8

Exemple — Taille des élèves EX

Taille X ~ N(160 cm, σ=10 cm). Calculer diverses probabilités.

On pose T = (X − 160)/10 ~ N(0,1)

P(X > 180) = P(T > 2) = 1 − Π(2) = 1 − 0,9772 = 0,0228
P(X ≤ 155) = P(T ≤ −0,5) = 1 − Π(0,5) = 1 − 0,6915 = 0,3085
P(169 ≤ X ≤ 178) = Π(1,8) − Π(0,9) = 0,9641 − 0,8159 = 0,1482

Exemple — Poids de pièces fabriquées EX

Poids X ~ N(768 g, σ²=6,25). σ = 2,5 g. Calculer P(X ≤ 770,95).

T = (770,95 − 768)/2,5 = 2,95/2,5 = 1,18
P(X ≤ 770,95) = Π(1,18) = 0,881

01 Tiercé avec 12 chevaux

Dénombrements
★★★
▼
Combien y a-t-il de tiercés dans l'ordre possibles avec 12 chevaux partants ?
Solution détaillée

Un tiercé dans l'ordre désigne les chevaux classés 1er, 2ème et 3ème. L'ordre intervient : c'est un arrangement de 3 éléments parmi 12.

N = A³₁₂ = 12 × 11 × 10 = 1 320

02 Nombres de 3 chiffres distincts

Dénombrements
★★★
▼
Combien peut-on former de nombres de 3 chiffres distincts à partir des 10 chiffres de la numérotation décimale ?
Solution détaillée

Nombre total d'arrangements d'ordre 3 parmi 10 :

A³₁₀ = 10 × 9 × 8 = 720

Mais parmi ceux-là, certains commencent par 0 (012, 013, …) et ne sont pas des nombres à 3 chiffres. Il faut les retrancher. Ce sont les arrangements d'ordre 2 parmi les 9 chiffres ≠ 0 :

A²₉ = 9 × 8 = 72
Réponse : 720 − 72 = 648

03 Placement sur un banc (filles et garçons)

Dénombrements
★★★
▼
4 filles et 3 garçons s'assoient sur un banc.
1. Les garçons sont côte à côte, les filles aussi.
2. Pas deux garçons ni deux filles côte à côte (alternance).
Solution détaillée

1. Groupes homogènes : On ordonne les 3 garçons entre eux (P₃ = 6 façons) et les 4 filles entre elles (P₄ = 24 façons). Les deux groupes peuvent être dans 2 ordres (G d'abord ou F d'abord).

N = 2 × 6 × 24 = 288

2. Alternance : Les filles (les plus nombreuses) occupent les places 1,3,5,7 et les garçons les places 2,4,6.

N = P₄ × P₃ = 24 × 6 = 144

04 Deux dés (rouge et bleu)

Probabilités
★★★
▼
On jette deux dés non pipés. Calculer :
a) P("obtenir 2 fois le point 1")
b) P("somme égale à 4")
c) P("somme égale à 11")

a) A₁ = {(1,1)} → P(A₁) = 1/36

05 Urne : 2 blanches, 3 noires — Espérance et variance

V.A. discrètes
★★★
▼
Une urne contient 5 boules dont 2 blanches et 3 noires. On tire une boule de l'urne sans la remettre, 3 fois de suite. X = nombre de boules blanches tirées. Calculer E(X), V(X) et σ(X).
Solution détaillée

X peut prendre les valeurs 0, 1, 2. On assimile à un tirage simultané de 3 boules parmi 5 (card Ω = C²₅ = 10).

P(X=0) = C³₃/C³₅ = 1/10   P(X=1) = C¹₂·C²₃/C³₅ = 6/10 = 3/5   P(X=2) = C²₂·C¹₃/C³₅ = 3/10
E(X) = 0×(1/10) + 1×(3/5) + 2×(3/10) = 0 + 6/10 + 6/10 = 6/5
V(X) = E(X²) − [E(X)]² = (0 + 3/5 + 12/10) − 36/25 = 9/5 − 36/25 = 9/25
σ(X) = √(9/25) = 3/5 = 0,6

06 Loi binomiale — Machines en panne

Binomiale/Poisson
★★★
▼
30 machines. Prob. de panne = 0,05 par jour. Z = nombre de pannes.
A parie qu'une seule machine tombera en panne ; B parie pour deux. Qui a plus de chances ?
Solution détaillée

Z ~ B(30 ; 0,05).

P(Z=1) = C¹₃₀ · (0,05)¹ · (0,95)²⁹ = 30 × 0,05 × (0,95)²⁹ ≈ 0,340
P(Z=2) = C²₃₀ · (0,05)² · (0,95)²⁸ ≈ 29/38 × P(Z=1) ≈ 0,259

P(Z=1) > P(Z=2) → A a le plus de chances de gagner.

07 Approximation de Poisson — Ruptures de stock

Binomiale/Poisson
★★★
▼
Prob. de rupture de stock hebdomadaire = 1,25%. X = nombre de ruptures en 52 semaines. Calculer P(X=0) et P(X>2) par approximation de Poisson.
Solution détaillée

n=52, p=0,0125. n≥30 et p≤0,1 → Approximation valide avec λ = np = 52×0,0125 = 0,65

P(X=0) = e⁻⁰·⁶⁵ ≈ 0,52
P(X>2) = 1 − [P(X=0)+P(X=1)+P(X=2)]
≈ 1 − [0,52 + 0,34 + 0,11] = 1 − 0,97 = 0,03

08 Loi normale — Filaments

Loi normale
★★★
▼
Diamètre X ~ N(150 mm, σ=5 mm). Tolérance ±8% par rapport à la moyenne. Calculer le pourcentage de rebuts.
Solution détaillée

Tolérance : 8% de 150 = 12 mm → Intervalle acceptable [138 ; 162].

Rebut si X ∉ [138 ; 162].

T = (X−150)/5 ~ N(0,1)
P(138 ≤ X ≤ 162) = P(−2,4 ≤ T ≤ 2,4) = 2Π(2,4)−1 = 2×0,9918−1 = 0,9836
P(rebut) = 1 − 0,9836 = 0,0164 → 1,64% de rebuts

09 Échantillonnage — Boulons défectueux

Loi normale
★★★
▼
4% des boulons sont défectueux. Caisses de 1000. Calculer :
1) P(proportion défectueux < 2%)
2) P(proportion défectueux > 5%)
Solution détaillée

p=0,04, n=1000. P suit N(p, √(pq/n)).

σ(P) = √(0,04×0,96/1000) ≈ 0,0062
1) P(P ≤ 0,02) = P(T ≤ (0,02−0,04)/0,0062) = P(T ≤ −3,22) = 1−Π(3,22) ≈ 0,065%
2) P(P ≥ 0,05) = P(T ≥ (0,05−0,04)/0,0062) = P(T ≥ 1,61) = 1−Π(1,61) ≈ 0,54%
